
# Social Buttons WordPress Plugin

## Plugin Information
- **Plugin Name:** Social Buttons
- **Description:** Add social buttons to your WordPress site to improve social media interaction.
- **Version:** 1.0
- **Author:** Team Six
- **Author URI:** http://www.example.com

---

## Installation
1. Download the plugin files and extract the ZIP archive.
2. Upload the extracted folder to the `/wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the WordPress Admin Panel under **Plugins > Installed Plugins**.
4. Configure the settings through the **Social Buttons** menu in the Admin Dashboard.

---

## Features
- Easily configurable social media buttons.
- Supports Facebook, WhatsApp, LinkedIn, and Twitter.
- Enter your social media profiles to link them to the buttons.
- Uses Font Awesome for icons and Bootstrap for responsive styling.

---

## Usage
1. Go to **Social Buttons** in the Admin Dashboard.
2. Enter the URLs for your social media profiles (Facebook, WhatsApp, LinkedIn, Twitter).
3. Save the settings.
4. The social media buttons will automatically appear on posts and pages.

---

## Files Overview

### Plugin Main File
`social-buttons.php`: The main file containing plugin logic and feature definitions.

### Admin Settings Page
`admin/sb-settings.php`: Generates the Admin Settings Page for entering social media links.

### Frontend Display
`frontend/sb-display.php`: Adds the social media buttons to your WordPress content.

### Assets
- `assets/css/style.css`: Custom styles for the plugin.
- `assets/bootstrap/css/bootstrap.min.css`: Bootstrap CSS.
- `assets/font-awesome/css/all.min.css`: Font Awesome for icons.
- `assets/bootstrap/js/bootstrap.bundle.min.js`: Bootstrap JS.
- `assets/js/main.js`: Custom JavaScript for the plugin.
